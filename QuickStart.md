# Prerequisites
| Item | Name | Version |
| - | - | - | 
| 1 | PowerShell | 5.1 or above |
| 2 | Az Module | 6.4.0 |
| 3 | Azure CLI | 2.28.0 |

# Procedure

1 Review **README.md** in **Connection**
1 Fork a repository or download the necessary script to local computer
1 Modify **Global Parameter**
1 Execute the script from Windows PowerShell
