# Azure Services Naming Principle

- Be entirely numeric
- Not contain the following characters: ` ~ ! @ # $ % ^ & * ( ) = + _ [ ] { } \ | ; : . ' " , < > / ?.
- Azure VM name cannot be more than 15 characters long
